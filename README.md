# 🌺 Uchinadix - Diccionario Uchinaguchi-Español

Buscador interactivo de términos en **Uchinaguchi** (lengua de Okinawa) y Castellano.

## 📖 Crédito y Fuente Cultural
Este proyecto fue desarrollado tomando como referencia la obra léxica del profesor **Munetaka Ganaha**, investigador de la cultura Ryu Kyu.

## 🛠️ Tecnologías
- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON
